布林邏輯基礎知識

在布林邏輯中，我們使用以下的符號表示基本運算：

  • ∧ 或 ∩：邏輯與（AND）或交集
  • ∨ 或 ∪：邏輯或（OR）或聯集
  • ¬ 或 ~：邏輯非（NOT）或補集
  • 0：邏輯假 / 空集
  • 1：邏輯真 / 全集

以下是布林代數中常見的運算律：

律名 表示式
結合律(a ∨ b) ∨ c = a ∨ (b ∨ c)
(a ∧ b) ∧ c = a ∧ (b ∧ c)
交換律a ∨ b = b ∨ a
a ∧ b = b ∧ a
分配律a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)
a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)
吸收律a ∨ (a ∧ b) = a
a ∧ (a ∨ b) = a
互補律a ∨ ¬a = 1
a ∧ ¬a = 0
冪等律a ∨ a = a
a ∧ a = a
有界律a ∨ 0 = a
a ∧ 1 = a
